# 🔧 دليل استخدام نظام الإدارة - UnlockVault

## 📋 نظرة عامة

تم إنشاء نظام إدارة متقدم يسمح بإضافة العروض تلقائياً بدون الحاجة لملفات JSON. النظام يستخدم localStorage للتخزين المحلي.

## 🚀 كيفية إضافة عروض جديدة

### 1. الوصول لصفحة الإدارة
- اذهب إلى: `http://localhost:3000/admin`
- ستجد نموذج شامل لإضافة العروض

### 2. ملء البيانات المطلوبة

#### المعلومات الأساسية:
- **العنوان**: اسم الأداة/اللعبة/التطبيق
- **Slug**: يتم توليده تلقائياً من العنوان
- **النوع**: اختر من (AI Tool, Game Hack, Premium App)
- **التصنيف**: يتغير حسب النوع المختار
- **الوصف**: وصف مفصل للعرض

#### رفع الصورة:
- اختر صورة من جهازك
- ستظهر معاينة للصورة
- يتم حفظ مسار الصورة تلقائياً

#### الميزات:
- أضف قائمة بميزات العرض
- يمكن إضافة/حذف الميزات ديناميكياً
- اضغط "+ Add Feature" لإضافة ميزة جديدة

#### روابط CPA:
- أضف روابط مختلفة حسب البلد
- مثال: us, gb, ca, au
- يمكن إضافة/حذف روابط متعددة

### 3. حفظ العرض
- اضغط "Add Item to Database"
- سيتم حفظ العرض في localStorage
- ستظهر رسالة تأكيد عند النجاح

## 🔍 كيفية عرض العروض

### في الصفحات الرئيسية:
- `/tools` - عرض جميع الأدوات
- `/games` - عرض جميع الألعاب  
- `/apps` - عرض جميع التطبيقات

### في صفحات التفاصيل:
- `/tools/[slug]` - تفاصيل الأداة
- `/games/[slug]` - تفاصيل اللعبة
- `/apps/[slug]` - تفاصيل التطبيق

### في البحث:
- `/search?q=keyword` - البحث الشامل
- تصفية حسب النوع (All, Tools, Games, Apps)

## 💾 نظام التخزين

### localStorage Keys:
- `unlockvault_tools` - الأدوات
- `unlockvault_games` - الألعاب  
- `unlockvault_apps` - التطبيقات

### البيانات الافتراضية:
- يتم تحميل بيانات افتراضية عند أول زيارة
- تشمل أمثلة لكل نوع من العروض

## 🔧 الميزات المتقدمة

### اكتشاف VPN:
- يتم فحص المستخدمين تلقائياً
- إعادة توجيه مستخدمي VPN لصفحة التحذير

### روابط CPA الذكية:
- اختيار الرابط حسب بلد المستخدم
- رابط احتياطي في حالة عدم وجود رابط للبلد

### العدّاد التنازلي:
- 5 دقائق لكل عرض
- يعمل في الوقت الفعلي

## 🛠️ استكشاف الأخطاء

### إذا لم تظهر العروض:
1. تأكد من تفعيل JavaScript
2. افحص console للأخطاء
3. امسح localStorage وأعد تحميل الصفحة

### إذا فشل حفظ العرض:
1. تأكد من ملء جميع الحقول المطلوبة
2. تأكد من اختيار تصنيف صحيح
3. تحقق من صحة روابط CPA

## 🔄 النسخ الاحتياطي

### لحفظ البيانات:
```javascript
// في console المتصفح
const tools = localStorage.getItem('unlockvault_tools');
const games = localStorage.getItem('unlockvault_games');  
const apps = localStorage.getItem('unlockvault_apps');

// احفظ هذه البيانات في ملف نصي
```

### لاستعادة البيانات:
```javascript
// في console المتصفح
localStorage.setItem('unlockvault_tools', 'البيانات_المحفوظة');
localStorage.setItem('unlockvault_games', 'البيانات_المحفوظة');
localStorage.setItem('unlockvault_apps', 'البيانات_المحفوظة');
```
